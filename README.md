# pLM4Alg Protocol: A protein language model based framework for allergen prediction.

This repository is intended to support the protocol, implementation workflow, and example usage of pLM4Alg for reproducible analysis.

---

## Overview

pLM4Alg is designed to predict allergenicity using protein sequence information encoded through pretrained protein language models and processed through deep learning based classification pipelines.

This repository provides:

- stepwise protocol guidance
- example workflow for data preparation
- sequence embedding generation
- model training and evaluation
- prediction on new sequences
- material that can support protocol reproduction and GitHub-based dissemination
